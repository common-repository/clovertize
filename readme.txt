== Description ==
Clovertize is a widget that lets users add clovers, leprechauns and other Irish clipart to your webpage. A fun addition for St. Patrick's Day. Includes several different styles for the Clovertize button or you can add your own button. 

Based on the code from Cornify.

== Installation ==
1. Download the plugin and extract the files
2. Upload "clovertize" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the Clovertize widget to and widget area
5. Add some attachments to a page or post

== Frequently Asked Questions ==

= Why would I add this to my site? =

Clovertize is meant to be fun and to get your readers to interact with your blog.

= The default buttons don't match my theme. What should I do? =

Clovertize can use a custom graphic for the button. Upload any image to WordPress, select the Custom button type and add a URL to the field in the widget box.
